# 4 - What is a quick way to create a new folder?
mkdir <folder name> in terminal.
# 5 - What output would you expect when you run the command 'ls' in an empty folder? What about a folder containing files?
In an empty folder there would be no output, while in a folder with 1 or more files/folders it will list the files and folders.

It shows you which changes have been saved by Git. It shows what Git can see as well.
# 9 - What does the command 'git add .' do?
